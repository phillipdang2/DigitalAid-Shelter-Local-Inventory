# Description

This is a lightweight full-stack web application built via React and Material UI on the front-end and Node.js, Express, and SQLite on the back-end, that allows users to input listings of donation information for a local shelter. 

# Deployment

To deploy this project run in the **backend** file directory the following:

```bash
  node server.js
```

and then run in the **frontend** file directory the following:

```bash
  npm start
```
